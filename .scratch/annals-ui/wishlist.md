# Annals UI — wishlist (deferred by decision, 2026-07-21)

Ideas explicitly deferred during the grilling session. Not tickets yet.

## Cross-linking dossiers

Faction and character names inside an event dossier become clickable, jumping
to that faction/character's dossier (and perhaps back). Deferred: a navigation
rabbit hole (link resolution, history/back behavior) that shouldn't ride along
on the first dossier ticket.

## Time-jump on event click

Clicking an event also scrubs the timeline to the event's year, so the map
shows the world *as it was* at the event.

**Hazard recorded:** the scrub cap hides events after the capped year, so the
click would truncate the very feed being read — the clicked row could vanish
mid-gesture. Needs a deliberate design (e.g. pin the feed during an event
jump) before it can be a ticket.
